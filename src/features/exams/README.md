# Provas e Simulados

Centro do módulo responsável por `exams`, `exam_attempts`, blueprints e resultados.

Pré-condições:

- as provas trimestrais e anuais precisarão de contadores (já expostos em `services/db`).
- os dashboards devem mostrar status e próximos exames.
- a camada de IA/edital governa as revisões e versões.
